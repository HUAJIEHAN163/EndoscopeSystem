## 一、原方案问题分析

### 1.1 严重Bug

#### Bug 1: LUT查找最大值错误（第89-91行）

**问题代码**:
```cpp
// ❌ 错误：只遍历前256个字节，应该遍历整个mask
uchar max_val = 0;
for (int i = 0; i < 256; i++) {
    if (mask.data[i] > max_val) max_val = mask.data[i];
}
```

**问题分析**:
- 320x240图像有76,800个像素，但只检查了前256个
- 导致归一化错误，mask值不准确
- 影响锐化效果的准确性

**修复方案**:
```cpp
// ✅ 正确：遍历所有像素
uchar max_val = 0;
const int total = mask.rows * mask.cols;
for (int i = 0; i < total; i++) {
    if (mask.data[i] > max_val) max_val = mask.data[i];
}
```

#### Bug 2: 混合循环中的除法性能问题（第129行）

**问题代码**:
```cpp
// ❌ 每次循环都做整数除法，非常慢（320x240 = 76,800次除法）
int result = src_val + (diff * mask_val) / 255;
```

**问题分析**:
- 整数除法在ARM Cortex-A7上需要多个周期
- 76,800次除法累计耗时显著
- 255可以近似为256，用位移替代

**修复方案**:
```cpp
// ✅ 用位移替代除法（255 ≈ 256，误差<0.4%）
int result = src_val + ((diff * mask_val) >> 8);
```

**性能提升**: 预计减少10-15%耗时

## 二、性能优化方案

### 2.1 优化1: NEON向量化混合循环（最重要）

**优化目标**: 混合循环是最大瓶颈，占总耗时40-50%

**优化策略**: 
- 利用项目现有NEON经验（参考 docs/13_9_NEON加速YUYV转BGR）
- 每次处理8个像素（24字节BGR）
- 使用NEON内置函数避免手动汇编

**实现代码**:
```cpp
// imageprocessor.cpp - NEON优化的混合函数
#ifdef __ARM_NEON
#include <arm_neon.h>
#endif

void ImageProcessor::blendWithMaskNEON(const uchar* src, const uchar* usm,
                                       const uchar* mask, uchar* dst,
                                       int total_pixels) {
    const int total = total_pixels * 3;  // BGR三通道
    int i = 0;
    
#ifdef __ARM_NEON
    // 每次处理8个像素（24字节）
    for (; i <= total - 24; i += 24) {
        // 加载源数据和USM数据（8像素 x 3通道）
        uint8x8_t src_0 = vld1_u8(src + i);
        uint8x8_t src_1 = vld1_u8(src + i + 8);
        uint8x8_t src_2 = vld1_u8(src + i + 16);
        
        uint8x8_t usm_0 = vld1_u8(usm + i);
        uint8x8_t usm_1 = vld1_u8(usm + i + 8);
        uint8x8_t usm_2 = vld1_u8(usm + i + 16);
        
        // 加载mask（8个像素，每个BGR共享1个mask值）
        uint8x8_t mask_vals = vld1_u8(mask + i / 3);
        
        // 计算差值：diff = usm - src（扩展到16位避免溢出）
        int16x8_t diff_0 = vreinterpretq_s16_u16(vsubl_u8(usm_0, src_0));
        int16x8_t diff_1 = vreinterpretq_s16_u16(vsubl_u8(usm_1, src_1));
        int16x8_t diff_2 = vreinterpretq_s16_u16(vsubl_u8(usm_2, src_2));
        
        // 扩展mask到16位
        uint16x8_t mask16 = vmovl_u8(mask_vals);
        int16x8_t mask16_s = vreinterpretq_s16_u16(mask16);
        
        // 计算：weighted = diff * mask
        int16x8_t weighted_0 = vmulq_s16(diff_0, mask16_s);
        int16x8_t weighted_1 = vmulq_s16(diff_1, mask16_s);
        int16x8_t weighted_2 = vmulq_s16(diff_2, mask16_s);
        
        // 右移8位（除以256）
        int16x8_t shifted_0 = vshrq_n_s16(weighted_0, 8);
        int16x8_t shifted_1 = vshrq_n_s16(weighted_1, 8);
        int16x8_t shifted_2 = vshrq_n_s16(weighted_2, 8);
        
        // 加回src：result = src + shifted
        int16x8_t result_0 = vaddq_s16(vreinterpretq_s16_u16(vmovl_u8(src_0)), shifted_0);
        int16x8_t result_1 = vaddq_s16(vreinterpretq_s16_u16(vmovl_u8(src_1)), shifted_1);
        int16x8_t result_2 = vaddq_s16(vreinterpretq_s16_u16(vmovl_u8(src_2)), shifted_2);
        
        // 饱和转换回uint8并存储
        vst1_u8(dst + i, vqmovun_s16(result_0));
        vst1_u8(dst + i + 8, vqmovun_s16(result_1));
        vst1_u8(dst + i + 16, vqmovun_s16(result_2));
    }
#endif
    
    // 处理剩余像素（标量代码）
    for (; i < total; i++) {
        int src_val = src[i];
        int usm_val = usm[i];
        int mask_val = mask[i / 3];
        int diff = usm_val - src_val;
        int result = src_val + ((diff * mask_val) >> 8);  // 使用位移
        dst[i] = static_cast<uchar>(std::max(0, std::min(255, result)));
    }
}
```

### 2.2 优化2: 内存预分配与复用

**优化目标**: 减少临时内存分配开销

**优化策略**:
- 基于项目现有零拷贝架构（参考 docs/13_10_FrameQueue零拷贝优化）
- 使用线程局部存储（thread_local）预分配缓冲区
- 避免每次调用都创建临时Mat对象

**实现代码**:
```cpp
// imageprocessor.h - 添加缓冲区结构
class ImageProcessor {
private:
    // 预分配缓冲区（线程局部存储）
    struct AdaptiveSharpenBuffers {
        cv::Mat gray;      // 灰度图
        cv::Mat lap;       // Laplacian结果
        cv::Mat mask;      // 边缘掩码
        cv::Mat blurred;   // 高斯模糊结果
        cv::Mat usm;       // USM锐化结果
        bool initialized = false;
        
        void init(const cv::Size& size) {
            if (!initialized || gray.size() != size) {
                gray.create(size, CV_8UC1);
                lap.create(size, CV_16S);
                mask.create(size, CV_8UC1);
                blurred.create(size, CV_8UC3);
                usm.create(size, CV_8UC3);
                initialized = true;
            }
        }
        
        void release() {
            gray.release();
            lap.release();
            mask.release();
            blurred.release();
            usm.release();
            initialized = false;
        }
    };
    
    static thread_local AdaptiveSharpenBuffers s_buffers;
    
public:
    // ... 现有方法声明
};

// imageprocessor.cpp - 定义线程局部变量
thread_local ImageProcessor::AdaptiveSharpenBuffers ImageProcessor::s_buffers;
```

### 2.3 优化3: YUV空间处理（效果优化）

**优化目标**: 避免色彩伪影，提升视觉效果

**优化策略**:
- 根据优化方案.md建议：始终在亮度通道（Y）处理
- 只对Y通道做锐化，UV通道保持不变
- 避免BGR空间直接锐化导致的颜色溢出

**实现代码**:
```cpp
// imageprocessor.cpp - YUV空间版本
void ImageProcessor::applyAdaptiveSharpenYUV(const cv::Mat &src, cv::Mat &dst,
                                             double amount, int threshold) {
    CV_Assert(src.type() == CV_8UC3);
    
    // 转换到YUV空间
    cv::Mat yuv;
    cv::cvtColor(src, yuv, cv::COLOR_BGR2YUV);
    
    // 分离通道
    std::vector<cv::Mat> channels;
    cv::split(yuv, channels);
    
    // 只对Y通道做自适应锐化
    cv::Mat y_sharpened;
    applyAdaptiveSharpenGray(channels[0], y_sharpened, amount, threshold);
    channels[0] = y_sharpened;
    
    // 合并通道并转回BGR
    cv::merge(channels, yuv);
    cv::cvtColor(yuv, dst, cv::COLOR_YUV2BGR);
}

// 灰度图版本（更快，省去cvtColor）
void ImageProcessor::applyAdaptiveSharpenGray(const cv::Mat &src, cv::Mat &dst,
                                              double amount, int threshold) {
    CV_Assert(src.type() == CV_8UC1);
    
    // 复用缓冲区
    s_buffers.init(src.size());
    
    // 直接在灰度图上计算Laplacian（省去BGR2GRAY转换）
    cv::Laplacian(src, s_buffers.lap, CV_16S, 1);
    
    // 生成mask
    cv::convertScaleAbs(s_buffers.lap, s_buffers.mask);
    
    // 阈值处理和归一化
    if (threshold > 0) {
        cv::threshold(s_buffers.mask, s_buffers.mask, threshold, 0, cv::THRESH_TOZERO);
        
        // 查找最大值（修复bug）
        uchar max_val = 0;
        const int total = s_buffers.mask.rows * s_buffers.mask.cols;
        const uchar* mask_data = s_buffers.mask.data;
        for (int i = 0; i < total; i++) {
            if (mask_data[i] > max_val) max_val = mask_data[i];
        }
        
        // LUT归一化
        if (max_val > 0) {
            uchar lut[256];
            for (int i = 0; i < 256; i++) {
                lut[i] = static_cast<uchar>((i * 255) / max_val);
            }
            cv::LUT(s_buffers.mask, cv::Mat(1, 256, CV_8U, lut), s_buffers.mask);
        }
    }
    
    // 计算USM（灰度图版本）
    cv::Mat blurred;
    cv::GaussianBlur(src, blurred, cv::Size(3, 3), 0);
    cv::addWeighted(src, 1.0 + amount, blurred, -amount, 0, s_buffers.usm);
    
    // 混合（灰度图只需处理1/3的数据）
    dst.create(src.size(), src.type());
    const int total = src.rows * src.cols;
    
    blendWithMaskNEON(src.data, s_buffers.usm.data, s_buffers.mask.data, 
                     dst.data, total);
}
```

**效果提升**: 消除色彩伪影，视觉效果更自然

### 2.4 优化4: 并行化处理（可选）

**优化目标**: 利用多核CPU提升性能

**优化策略**:
- 使用OpenCV的parallel_for_框架
- 按行并行处理混合循环
- 适合图像编辑页离线处理（不阻塞UI）

**实现代码**:
```cpp
// imageprocessor.cpp - 并行混合类
class AdaptiveBlendParallel : public cv::ParallelLoopBody {
private:
    const uchar* src_;
    const uchar* usm_;
    const uchar* mask_;
    uchar* dst_;
    int width_;
    
public:
    AdaptiveBlendParallel(const uchar* src, const uchar* usm,
                         const uchar* mask, uchar* dst, int width)
        : src_(src), usm_(usm), mask_(mask), dst_(dst), width_(width) {}
    
    void operator()(const cv::Range& range) const override {
        for (int row = range.start; row < range.end; row++) {
            const int offset = row * width_ * 3;
            const int mask_offset = row * width_;
            
            // 使用NEON处理这一行
            ImageProcessor::blendWithMaskNEON(
                src_ + offset, usm_ + offset,
                mask_ + mask_offset, dst_ + offset, width_);
        }
    }
};

// 在主函数中使用
void ImageProcessor::applyAdaptiveSharpenParallel(const cv::Mat &src, cv::Mat &dst,
                                                  double amount, int threshold) {
    // ... 前面的Laplacian、mask生成、USM计算 ...
    
    // 并行混合
    dst.create(src.size(), src.type());
    cv::parallel_for_(cv::Range(0, src.rows),
                     AdaptiveBlendParallel(src.data, usm.data, mask.data,
                                          dst.data, src.cols));
}
```

**实现代码**:

```cpp
void ImageProcessor::applyAdaptiveSharpenFast(const cv::Mat &src, cv::Mat &dst,
                                              double amount, int threshold) {
    // 缩小到1/2分辨率
    cv::Mat gray_small, lap_small, mask_small;
    cv::Mat gray;
    cv::cvtColor(src, gray, cv::COLOR_BGR2GRAY);
    cv::resize(gray, gray_small, cv::Size(), 0.5, 0.5, cv::INTER_LINEAR);
    
    // 在小图上计算Laplacian
    cv::Laplacian(gray_small, lap_small, CV_16S, 1);
    cv::convertScaleAbs(lap_small, mask_small);
    
    // 阈值和归一化
    if (threshold > 0) {
        cv::threshold(mask_small, mask_small, threshold/2, 0, cv::THRESH_TOZERO);
        // ... LUT归一化
    }
    
    // 上采样mask到原始分辨率
    cv::Mat mask;
    cv::resize(mask_small, mask, src.size(), 0, 0, cv::INTER_LINEAR);
    
    // 后续USM和混合步骤不变
    // ...
}
```

```cpp
// imageprocessor.cpp
#ifdef __ARM_NEON
#include <arm_neon.h>
#endif

// 定义线程局部缓冲区
thread_local ImageProcessor::AdaptiveSharpenBuffers ImageProcessor::s_buffers;

// ========== 主函数：性能优化版 ==========
void ImageProcessor::applyAdaptiveSharpenOpt(const cv::Mat &src, cv::Mat &dst,
                                             double amount, int threshold) {
    // 参数验证
    if (src.empty() || src.type() != CV_8UC3) {
        qWarning() << "[自适应锐化] 输入图像格式错误";
        src.copyTo(dst);
        return;
    }
    
    amount = std::clamp(amount, 0.1, 5.0);
    threshold = std::clamp(threshold, 0, 255);
    
    QElapsedTimer timer;
    timer.start();
    
    // 初始化缓冲区
    s_buffers.init(src.size());
    
    // === 步骤1: 计算Laplacian ===
    cv::cvtColor(src, s_buffers.gray, cv::COLOR_BGR2GRAY);
    cv::Laplacian(s_buffers.gray, s_buffers.lap, CV_16S, 1);
    cv::convertScaleAbs(s_buffers.lap, s_buffers.mask);
    
    qint64 t_laplacian = timer.restart();
    
    // === 步骤2: 阈值和归一化 ===
    if (threshold > 0) {
        cv::threshold(s_buffers.mask, s_buffers.mask, threshold, 0, cv::THRESH_TOZERO);
        
        // 查找最大值（修复bug）
        uchar max_val = 0;
        const int total = s_buffers.mask.rows * s_buffers.mask.cols;
        const uchar* mask_data = s_buffers.mask.data;
        for (int i = 0; i < total; i++) {
            if (mask_data[i] > max_val) max_val = mask_data[i];
        }
        
        // LUT归一化
        if (max_val > 0) {
            uchar lut[256];
            for (int i = 0; i < 256; i++) {
                lut[i] = static_cast<uchar>((i * 255) / max_val);
            }
            cv::LUT(s_buffers.mask, cv::Mat(1, 256, CV_8U, lut), s_buffers.mask);
        }
    }
    
    qint64 t_mask = timer.restart();
    
    // === 步骤3: 计算USM ===
    cv::GaussianBlur(src, s_buffers.blurred, cv::Size(3, 3), 0);
    cv::addWeighted(src, 1.0 + amount, s_buffers.blurred, -amount, 0, s_buffers.usm);
    
    qint64 t_usm = timer.restart();
    
    // === 步骤4: NEON混合 ===
    dst.create(src.size(), src.type());
    const int total_pixels = src.rows * src.cols;
    
    blendWithMaskNEON(src.data, s_buffers.usm.data, s_buffers.mask.data,
                     dst.data, total_pixels);
    
    qint64 t_blend = timer.elapsed();
    
    qDebug() << QString("[自适应锐化v2] Lap:%1ms Mask:%2ms USM:%3ms Blend:%4ms Total:%5ms")
                .arg(t_laplacian).arg(t_mask).arg(t_usm).arg(t_blend)
                .arg(t_laplacian + t_mask + t_usm + t_blend);
}

// ========== NEON混合函数 ==========
void ImageProcessor::blendWithMaskNEON(const uchar* src, const uchar* usm,
                                       const uchar* mask, uchar* dst,
                                       int total_pixels) {
    const int total = total_pixels * 3;
    int i = 0;
    
#ifdef __ARM_NEON
    // 每次处理8个像素（24字节BGR）
    for (; i <= total - 24; i += 24) {
        uint8x8_t src_0 = vld1_u8(src + i);
        uint8x8_t src_1 = vld1_u8(src + i + 8);
        uint8x8_t src_2 = vld1_u8(src + i + 16);
        
        uint8x8_t usm_0 = vld1_u8(usm + i);
        uint8x8_t usm_1 = vld1_u8(usm + i + 8);
        uint8x8_t usm_2 = vld1_u8(usm + i + 16);
        
        uint8x8_t mask_vals = vld1_u8(mask + i / 3);
        
        int16x8_t diff_0 = vreinterpretq_s16_u16(vsubl_u8(usm_0, src_0));
        int16x8_t diff_1 = vreinterpretq_s16_u16(vsubl_u8(usm_1, src_1));
        int16x8_t diff_2 = vreinterpretq_s16_u16(vsubl_u8(usm_2, src_2));
        
        int16x8_t mask16_s = vreinterpretq_s16_u16(vmovl_u8(mask_vals));
        
        int16x8_t weighted_0 = vmulq_s16(diff_0, mask16_s);
        int16x8_t weighted_1 = vmulq_s16(diff_1, mask16_s);
        int16x8_t weighted_2 = vmulq_s16(diff_2, mask16_s);
        
        int16x8_t shifted_0 = vshrq_n_s16(weighted_0, 8);
        int16x8_t shifted_1 = vshrq_n_s16(weighted_1, 8);
        int16x8_t shifted_2 = vshrq_n_s16(weighted_2, 8);
        
        int16x8_t result_0 = vaddq_s16(vreinterpretq_s16_u16(vmovl_u8(src_0)), shifted_0);
        int16x8_t result_1 = vaddq_s16(vreinterpretq_s16_u16(vmovl_u8(src_1)), shifted_1);
        int16x8_t result_2 = vaddq_s16(vreinterpretq_s16_u16(vmovl_u8(src_2)), shifted_2);
        
        vst1_u8(dst + i, vqmovun_s16(result_0));
        vst1_u8(dst + i + 8, vqmovun_s16(result_1));
        vst1_u8(dst + i + 16, vqmovun_s16(result_2));
    }
#endif
    
    // 处理剩余像素
    for (; i < total; i++) {
        int src_val = src[i];
        int usm_val = usm[i];
        int mask_val = mask[i / 3];
        int diff = usm_val - src_val;
        int result = src_val + ((diff * mask_val) >> 8);
        dst[i] = static_cast<uchar>(std::max(0, std::min(255, result)));
    }
}
```
