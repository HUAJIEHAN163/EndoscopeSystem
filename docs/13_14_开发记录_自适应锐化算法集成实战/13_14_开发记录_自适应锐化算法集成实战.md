# 13_14 开发记录：自适应锐化算法集成实战（P14）

## 一、背景

### 1.1 当前锐化实现的问题

当前 `applySharpen` 是标准 USM（Unsharp Masking）：

```cpp
void ImageProcessor::applySharpen(const cv::Mat &src, cv::Mat &dst,
                                  double sigma, double amount) {
    cv::Mat blurred;
    cv::GaussianBlur(src, blurred, cv::Size(3, 3), 0);
    cv::addWeighted(src, 1.0 + amount, blurred, -amount, 0, dst);
}
```

**基准性能**（320x240，Cortex-A7）：~7ms/帧

**已知问题**：
1. **对噪声敏感**：平坦区域的噪声也会被锐化放大
2. **一视同仁**：对所有区域用同样强度，内窥镜场景中更希望只锐化边缘
3. **彩色伪影**：在 BGR 空间直接锐化可能导致边缘处颜色溢出

### 1.2 内窥镜场景的特殊需求

内窥镜图像特点：
- **平坦区域多**：黏膜背景相对平滑，不需要强锐化
- **边缘区域重要**：病灶轮廓、组织边界需要清晰增强
- **噪声敏感**：低光环境下噪声明显，应避免放大

### 1.3 目标算法：自适应边缘锐化

**设计思路**：
```
1. 计算每个像素的局部梯度强度（Laplacian 或 Sobel）
2. 梯度强度 → 锐化权重掩码（梯度大的地方权重高）
3. 只在权重高的区域叠加 USM 锐化结果
```

**与 USM 的对比**：

| | 标准 USM | 自适应边缘锐化 |
|---|---|---|
| 平坦区域 | 轻微增强（可能放大噪声） | 保持原样 |
| 边缘区域 | 均匀增强 | 按梯度强度增强 |
| 噪声敏感性 | 高 | 低（噪声梯度弱，权重低） |
| 计算量 | 低 | 中（多一次 Laplacian + normalize） |

## 二、设计方案

### 2.1 算法原理

**伪代码**：
```
// 1. 计算边缘掩码
gray = cvtColor(src, BGR2GRAY)
lap = Laplacian(gray, CV_32F, 1)      // 计算二阶导数，对边缘敏感
mask = abs(lap)                       // 取绝对值
mask = normalize(mask, 0, 1)          // 归一化到 [0,1]

// 2. 计算 USM 结果
blurred = GaussianBlur(src, Size(3,3), 0)
usm = src * (1.0 + amount) - blurred * amount

// 3. 按掩码混合
dst = src * (1 - mask) + usm * mask   // 边缘处用 USM，平坦处用原图
```

**数学表达**：
```
mask(x,y) = |∇²I_gray(x,y)| / max(|∇²I_gray|)
dst(x,y) = I_src(x,y) * (1 - mask(x,y)) + I_usm(x,y) * mask(x,y)
```

### 2.2 为什么选择 Laplacian

| 算子 | 优点 | 缺点 | 选择理由 |
|------|------|------|----------|
| **Laplacian** | 对边缘敏感，计算简单 | 对噪声敏感 | 内窥镜图像相对干净，噪声可控 |
| **Sobel** | 抗噪性好 | 只检测特定方向边缘 | 需要计算 x/y 两个方向，更复杂 |
| **Canny** | 边缘检测准确 | 计算量大，需要阈值调参 | 过于复杂，不适合实时处理 |

**Laplacian 的适应性**：
- 内窥镜图像噪声相对较少（LED 照明充足）
- Laplacian 能很好地区分平坦区域和边缘区域
- 计算量适中，适合实时处理

**方案选择**：
1. **替换现有 USM**：直接修改 `applySharpen`，风险高，影响实时流
2. **新增独立算法**：新增 `applyAdaptiveSharpen`，与 USM 并存，更灵活

**选择方案 2**：新增独立算法，原因：
- 不影响现有实时流的稳定性
- 可以在图像编辑页提供两种锐化选项对比
- 便于性能对比和效果评估

```cpp
// 简化版算法声明（快速学习版）
static void applyAdaptiveSharpenSimple(const cv::Mat &src, cv::Mat &dst,
                                       double amount = 1.5);

// 简化实现：只包含核心逻辑，便于理解和调试
void ImageProcessor::applyAdaptiveSharpenSimple(const cv::Mat &src, cv::Mat &dst,
                                                double amount) {
    // 1. 计算边缘掩码（简化：无阈值，直接归一化）
    cv::Mat gray, lap, mask;
    cv::cvtColor(src, gray, cv::COLOR_BGR2GRAY);
    cv::Laplacian(gray, lap, CV_16S, 1);  // 使用short类型，避免浮点
    cv::convertScaleAbs(lap, mask);       // |Laplacian| → [0, 255]
    
    // 简单归一化：mask = mask / max(mask)
    double maxVal;
    cv::minMaxLoc(mask, nullptr, &maxVal);
    if (maxVal > 0) mask.convertTo(mask, CV_32F, 1.0 / maxVal);
    
    // 2. 计算USM（复用现有优化）
    cv::Mat blurred, usm;
    cv::GaussianBlur(src, blurred, cv::Size(3, 3), 0);
    cv::addWeighted(src, 1.0 + amount, blurred, -amount, 0, usm);
    
    // 3. 手动混合循环（便于理解和调试）
    dst.create(src.size(), src.type());
    for (int y = 0; y < src.rows; y++) {
        const uchar* src_row = src.ptr<uchar>(y);
        const uchar* usm_row = usm.ptr<uchar>(y);
        const float* mask_row = mask.ptr<float>(y);
        uchar* dst_row = dst.ptr<uchar>(y);
        
        for (int x = 0; x < src.cols; x++) {
            float alpha = mask_row[x];  // 边缘强度权重
            for (int c = 0; c < 3; c++) {
                int idx = x * 3 + c;
                // 线性混合：dst = src*(1-alpha) + usm*alpha
                int val = src_row[idx] * (1 - alpha) + usm_row[idx] * alpha;
                dst_row[idx] = cv::saturate_cast<uchar>(val);
            }
        }
    }
}
```

### 3.3 阶段二：性能优化版实施计划（2天）

#### Day 4：性能分析与优化（4小时）
**目标**：分析性能瓶颈，实施关键优化，实践优化方法

**性能瓶颈分析**（基于学习版）：
1. **浮点运算瓶颈**：mask使用float，混合使用浮点乘法
2. **内存访问瓶颈**：手动循环效率低，缓存不友好
3. **临时变量开销**：多次创建临时Mat对象

**优化方案实施**：
```cpp
// 性能优化版
void ImageProcessor::applyAdaptiveSharpenOpt(const cv::Mat &src, cv::Mat &dst,
                                             double amount, int threshold = 10) {
    // 优化1：整数Laplacian，避免浮点转换
    cv::Mat gray, lap;
    cv::cvtColor(src, gray, cv::COLOR_BGR2GRAY);
    cv::Laplacian(gray, lap, CV_16S, 1);  // 使用short，计算更快
    
    // 优化2：整数mask处理
    cv::Mat mask;
    cv::convertScaleAbs(lap, mask);  // |Laplacian| → uint8
    
    // 整数阈值处理（避免浮点比较）
    if (threshold > 0) {
        cv::threshold(mask, mask, threshold, 0, cv::THRESH_TOZERO);
    }
    
    // 优化3：查找表加速归一化
    uchar lut[256];
    uchar max_val = 0;
    const uchar* mask_data = mask.data;
    const int total_pixels = mask.rows * mask.cols;
    
    // 找到最大值
    for (int i = 0; i < total_pixels; i++) {
        if (mask_data[i] > max_val) max_val = mask_data[i];
    }
    
    // 预计算归一化查找表
    if (max_val > 0) {
        for (int i = 0; i < 256; i++) {
            lut[i] = static_cast<uchar>((i * 255) / max_val);
        }
        cv::LUT(mask, cv::Mat(1, 256, CV_8U, lut), mask);
    }
    
    // 优化4：高效USM计算（复用现有优化）
    cv::Mat blurred, usm;
    cv::GaussianBlur(src, blurred, cv::Size(3, 3), 0);
    cv::addWeighted(src, 1.0 + amount, blurred, -amount, 0, usm);
    
    // 优化5：优化混合循环（整数运算，减少分支）
    dst.create(src.size(), src.type());
    const int total = src.rows * src.cols * 3;
    const uchar* src_data = src.data;
    const uchar* usm_data = usm.data;
    const uchar* mask_data_final = mask.data;
    uchar* dst_data = dst.data;
    
    // 展开循环，减少分支预测失败
    for (int i = 0; i < total; i += 3) {
        int mask_idx = i / 3;
        uchar alpha = mask_data_final[mask_idx];
        
        // 处理BGR三个通道
        for (int c = 0; c < 3; c++) {
            int idx = i + c;
            int diff = usm_data[idx] - src_data[idx];
            int result = src_data[idx] + (diff * alpha) / 255;
            
            // 快速钳位：result = min(max(result, 0), 255)
            dst_data[idx] = result < 0 ? 0 : (result > 255 ? 255 : result);
        }
    }
}
```

### 3.2 阶段 2：算法实现与单元测试（1天）

#### 3.2.1 在 `imageprocessor.cpp` 中实现

```cpp
void ImageProcessor::applyAdaptiveSharpen(const cv::Mat &src, cv::Mat &dst,
                                          double amount, int threshold) {
    CV_Assert(src.type() == CV_8UC3);  // 必须是 BGR 格式
    
    // 1. 计算边缘掩码（在灰度图上做 Laplacian）
    cv::Mat gray, lap;
    cv::cvtColor(src, gray, cv::COLOR_BGR2GRAY);
    cv::Laplacian(gray, lap, CV_32F, 1);  // 使用 float 避免溢出
    
    // 2. 取绝对值并归一化
    cv::Mat mask;
    cv::convertScaleAbs(lap, mask);  // |Laplacian|，范围 [0,255]
    mask.convertTo(mask, CV_32F, 1.0 / 255.0);  // 归一化到 [0,1]
    
    // 3. 阈值截断（可选，根据 threshold 参数决定）
    if (threshold > 0) {
        float thresh = threshold / 255.0f;
        cv::threshold(mask, mask, thresh, 0, cv::THRESH_TOZERO);
        // 重新归一化到 [0,1]
        double minVal, maxVal;
        cv::minMaxLoc(mask, &minVal, &maxVal);
        if (maxVal > 0) mask /= maxVal;
    }
    
    // 4. 计算 USM 结果
    cv::Mat blurred, usm;
    cv::GaussianBlur(src, blurred, cv::Size(3, 3), 0);
    cv::addWeighted(src, 1.0 + amount, blurred, -amount, 0, usm);
    
    // 5. 按掩码混合
    // 将 mask 从单通道扩展为三通道
    std::vector<cv::Mat> maskChannels = {mask, mask, mask};
    cv::Mat mask3;
    cv::merge(maskChannels, mask3);
    
    // 浮点混合：dst = src * (1-mask) + usm * mask
    cv::Mat srcF, usmF, dstF;
    src.convertTo(srcF, CV_32FC3, 1.0 / 255.0);
    usm.convertTo(usmF, CV_32FC3, 1.0 / 255.0);
    
    dstF = srcF.mul(1.0f - mask3) + usmF.mul(mask3);
    dstF.convertTo(dst, CV_8UC3, 255.0);
}
```

#### 3.4.2 优化方案

**优化 A：用整数运算替代浮点混合（最有效）**

```cpp
// 优化前：浮点 mul + merge（~8ms）
dstF = srcF.mul(1.0f - mask3) + usmF.mul(mask3);

// 优化后：mask 量化为 uint8，用 addWeighted（~2ms）
cv::Mat mask8;
mask.convertTo(mask8, CV_8U, 255.0);

// 对每个通道分别混合
cv::Mat channels[3];
cv::split(src, channels);
cv::Mat usmChannels[3];
cv::split(usm, usmChannels);

for (int i = 0; i < 3; i++) {
    cv::addWeighted(channels[i], 1.0, usmChannels[i], 1.0, 0, channels[i]);
    // 实际上需要更精细的混合，这里简化示意
}
cv::merge(channels, 3, dst);
```

// 将 mask 缩小再放大，相当于块平均
