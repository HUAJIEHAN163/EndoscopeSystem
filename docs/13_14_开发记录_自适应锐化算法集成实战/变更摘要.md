```cpp
void ImageProcessor::applyAdaptiveSharpen(const cv::Mat &src, cv::Mat &dst,
                                          double amount, int threshold) {
    // 1. 转换为灰度图计算边缘强度
    cv::Mat gray;
    cv::cvtColor(src, gray, cv::COLOR_BGR2GRAY);

    // 2. 计算 Laplacian 边缘强度
    cv::Mat laplacian;
    cv::Laplacian(gray, laplacian, CV_16S, 3);
    cv::convertScaleAbs(laplacian, laplacian);

    // 3. 归一化边缘强度到 0-255
    double minVal, maxVal;
    cv::minMaxLoc(laplacian, &minVal, &maxVal);
    cv::Mat edgeMask;
    if (maxVal > 0) {
        laplacian.convertTo(edgeMask, CV_32F, 255.0 / maxVal);
    } else {
        edgeMask = cv::Mat::zeros(laplacian.size(), CV_32F);
    }

    // 4. 应用阈值过滤
    float thresholdNorm = threshold * 255.0f / 50.0f;
    edgeMask = cv::max(edgeMask - thresholdNorm, 0.0f);
    
    // 重新归一化到 0-1 范围
    cv::minMaxLoc(edgeMask, &minVal, &maxVal);
    if (maxVal > 0) {
        edgeMask /= maxVal;
    }

    // 5. 计算 USM 锐化结果
    cv::Mat blurred, sharpened;
    cv::GaussianBlur(src, blurred, cv::Size(3, 3), 0);
    cv::addWeighted(src, 1.0 + amount, blurred, -amount, 0, sharpened);

    // 6. 根据边缘掩码混合原图和锐化图
    src.convertTo(dst, CV_32FC3);
    sharpened.convertTo(sharpened, CV_32FC3);

    // 将单通道掩码扩展为三通道
    std::vector<cv::Mat> maskChannels(3);
    maskChannels[0] = edgeMask;
    maskChannels[1] = edgeMask;
    maskChannels[2] = edgeMask;
    cv::Mat mask3ch;
    cv::merge(maskChannels, mask3ch);

    // 混合：dst = dst + (sharpened - dst) × mask3ch
    cv::Mat diff = sharpened - dst;
    diff = diff.mul(mask3ch);
    dst = dst + diff;

    // 转换回 CV_8UC3
    dst.convertTo(dst, CV_8UC3);
}
```

## 核心功能说明

### 1. 自适应锐化算法原理
- 使用 **Laplacian 算子**检测图像边缘
- 根据边缘强度生成**自适应掩码**
- 只对边缘区域应用锐化，平滑区域保持不变
- 避免了传统USM锐化的过度锐化和噪声放大问题

### 2. 互斥控制
- USM锐化和自适应锐化**同时只能启用一个**
- 勾选一个时自动取消另一个
- 防止重复锐化导致的画质劣化

### 3. 参数说明
| 参数 | 范围 | 默认值 | 说明 |
|------|------|--------|------|
| 锐化强度 | 0.5-3.0 | 1.5 | 控制锐化效果的强弱 |
| 边缘阈值 | 0-50 | 10 | 控制哪些区域被认为是边缘 |

## 使用方法

1. 启动程序后切换到**图像编辑模式**
2. 点击"图像选择"加载要处理的图像
3. 勾选"自适应锐化"复选框
4. 调整"强度"和"边缘阈值"滑块查看实时预览
5. 点击"保存"按钮保存处理后的图像
