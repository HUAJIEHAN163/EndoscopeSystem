# 自适应锐化参数详解

## 参数概览

自适应锐化有两个关键参数：

```cpp
void applyAdaptiveSharpen(const cv::Mat &src, cv::Mat &dst,
                          double amount,      // 锐化强度
                          int threshold);     // 边缘阈值
```

## 参数1：锐化强度（amount）

### 作用
控制锐化的**强度**，即对检测到的边缘区域施加多大的锐化效果。

### 工作原理

在USM锐化公式中：
```cpp
sharpened = src + amount × (src - blurred)
```

- `amount = 0.0`：无锐化效果
- `amount = 0.5`：轻微锐化
- `amount = 1.0`：中等锐化
- `amount = 1.5`：强锐化
- `amount = 2.0`：非常强的锐化

### 视觉效果

```
amount = 0.5          amount = 1.0          amount = 2.0
边缘轻微增强          边缘明显增强          边缘非常锐利
细节提升小            细节提升中等          细节提升大
光晕风险低            光晕风险中等          光晕风险高
```

### 推荐值

- **内窥镜组织图像**：1.0 - 1.5（需要较强增强）
- **高对比度图像**：0.5 - 1.0（避免过度锐化）
- **调试起点**：1.0

### 代码中的使用

```cpp
// 在自适应锐化中
cv::addWeighted(src, 1.0, unsharpMask, amount, 0, sharpened);
//                          ^^^^^^ 这里控制锐化强度
```

## 参数2：边缘阈值（threshold）

### 作用
控制**哪些区域**会被锐化，即决定"边缘"的定义标准。

### 工作原理

1. **边缘检测**：使用Laplacian算子计算每个像素的边缘强度（0-255）
2. **阈值过滤**：只保留边缘强度 > threshold 的区域
3. **生成掩码**：这些区域的掩码值为1，其他区域为0
4. **选择性锐化**：只对掩码值>0的区域应用锐化

### 阈值范围

UI中：`0 - 50`（映射到内部的 0-255）

```cpp
float thresholdNorm = threshold * 255.0f / 50.0f;
edgeMask = cv::max(edgeMask - thresholdNorm, 0.0f);
```

### 视觉效果

```
threshold = 0           threshold = 20          threshold = 40
所有区域都锐化         只锐化中等边缘          只锐化强边缘
效果接近USM锐化        平衡效果                保守锐化
可能有光晕             光晕较少                几乎无光晕
```

### 边缘强度示例

假设图像中不同区域的Laplacian边缘强度：

```
区域类型              边缘强度    threshold=0  threshold=20  threshold=40
-------------------- ----------- ------------ ------------- -------------
平滑组织（无边缘）    5           锐化 ✓       不锐化 ✗      不锐化 ✗
轻微纹理              15          锐化 ✓       不锐化 ✗      不锐化 ✗
中等边缘（血管）      30          锐化 ✓       锐化 ✓        不锐化 ✗
强边缘（器官轮廓）    50          锐化 ✓       锐化 ✓        锐化 ✓
极强边缘（文字）      80          锐化 ✓       锐化 ✓        锐化 ✓
```

### 推荐值

- **threshold = 0-10**：锐化所有区域（类似USM，可能有光晕）
- **threshold = 15-25**：锐化中等以上边缘（推荐，平衡效果）
- **threshold = 30-40**：只锐化强边缘（保守，适合高对比度图像）
- **threshold = 45-50**：几乎不锐化（仅极强边缘）

### 代码中的使用

```cpp
// 1. 计算边缘强度
cv::Laplacian(gray, laplacian, CV_16S, 3);
cv::convertScaleAbs(laplacian, laplacian);  // 0-255

// 2. 应用阈值过滤
float thresholdNorm = threshold * 255.0f / 50.0f;
edgeMask = cv::max(edgeMask - thresholdNorm, 0.0f);
//                            ^^^^^^^^^^^^^^ 低于阈值的区域变为0

// 3. 归一化为0-1的混合权重
if (maxVal > 0) {
    edgeMask /= maxVal;  // 强边缘=1.0，弱边缘=0.0-1.0之间
}
```

## 两个参数的协同作用

### 组合效果矩阵

```
                    threshold低（0-10）      threshold中（15-25）    threshold高（30-40）
amount低（0.5-1.0） 轻微全局锐化            轻微边缘锐化            几乎无效果
amount中（1.0-1.5） 中等全局锐化 ⭐         中等边缘锐化 ⭐⭐       轻微边缘锐化
amount高（1.5-2.0） 强全局锐化（有光晕）    强边缘锐化 ⭐          中等边缘锐化
```

⭐⭐ = 推荐组合
⭐ = 可用组合

### 典型应用场景

#### 场景1：内窥镜组织图像（整体模糊）
```
amount = 1.5        // 需要强锐化
threshold = 15      // 锐化大部分区域，但避免平滑区域
效果：组织细节明显增强，血管清晰，无明显光晕
```

#### 场景2：带文字的内窥镜图像
```
amount = 1.0        // 中等锐化
threshold = 25      // 提高阈值，避免文字边缘过度锐化
效果：组织细节增强，文字光晕减少
```

#### 场景3：高对比度图像（如X光片）
```
amount = 0.8        // 轻微锐化
threshold = 30      // 只锐化强边缘
效果：边缘清晰，无光晕
```

## 调参建议

### 调参顺序

1. **先调threshold**（决定锐化区域）
   - 从20开始
   - 如果光晕明显 → 增加threshold
   - 如果效果不明显 → 降低threshold

2. **再调amount**（决定锐化强度）
   - 从1.0开始
   - 如果细节不够清晰 → 增加amount
   - 如果有光晕或过度锐化 → 降低amount

### 调参技巧

**问题：锐化效果不明显**
- 原因1：threshold太高 → 降低threshold
- 原因2：amount太低 → 增加amount
- 原因3：图像本身边缘弱 → 降低threshold

**问题：有明显光晕**
- 原因1：threshold太低 → 增加threshold（优先）
- 原因2：amount太高 → 降低amount
- 原因3：高对比度区域 → 增加threshold到30+

**问题：部分区域过度锐化**
- 增加threshold，让算法更有选择性

## 与USM锐化的对比

### USM锐化（applySharpen）
```cpp
applySharpen(src, dst, sigma, amount);
```
- **sigma**：控制模糊范围（固定，用户不可调）
- **amount**：控制锐化强度
- **特点**：全局锐化，所有区域一视同仁

### 自适应锐化（applyAdaptiveSharpen）
```cpp
applyAdaptiveSharpen(src, dst, amount, threshold);
```
- **amount**：控制锐化强度（与USM相同）
- **threshold**：控制锐化区域（新增）
- **特点**：选择性锐化，根据边缘强度自动调整

### 选择建议

| 图像类型           | 推荐算法       | 原因                           |
|-------------------|---------------|--------------------------------|
| 内窥镜组织图像     | 自适应锐化 ⭐⭐ | 避免平滑区域噪声放大            |
| 带文字的图像       | 自适应锐化 ⭐⭐ | 自动避免文字边缘光晕            |
| 高对比度图像       | 自适应锐化 ⭐  | 可控制锐化区域                  |
| 简单场景          | USM锐化 ⭐     | 更简单，参数少                  |

## 实际代码流程

```cpp
void ImageProcessor::applyAdaptiveSharpen(const cv::Mat &src, cv::Mat &dst,
                                          double amount, int threshold) {
    // === 步骤1：边缘检测 ===
    cv::Mat gray;
    cv::cvtColor(src, gray, cv::COLOR_BGR2GRAY);
    cv::Mat laplacian;
    cv::Laplacian(gray, laplacian, CV_16S, 3);
    cv::convertScaleAbs(laplacian, laplacian);  // 边缘强度 0-255
    
    // === 步骤2：生成边缘掩码 ===
    cv::Mat edgeMask;
    laplacian.convertTo(edgeMask, CV_32F, 255.0 / maxVal);  // 归一化
    
    float thresholdNorm = threshold * 255.0f / 50.0f;
    edgeMask = cv::max(edgeMask - thresholdNorm, 0.0f);  // 应用阈值
    //                            ^^^^^^^^^^^^^^
    //                            threshold参数在这里起作用
    
    edgeMask /= maxVal;  // 归一化到0-1，作为混合权重
    
    // === 步骤3：计算USM锐化 ===
    cv::Mat blurred, unsharpMask, sharpened;
    cv::GaussianBlur(src, blurred, cv::Size(3, 3), 0.8);
    cv::subtract(src, blurred, unsharpMask);
    cv::addWeighted(src, 1.0, unsharpMask, amount, 0, sharpened);
    //                                     ^^^^^^
    //                                     amount参数在这里起作用
    
    // === 步骤4：根据掩码混合 ===
    // result = src + (sharpened - src) × edgeMask
    //                                    ^^^^^^^^^
    //                                    掩码控制每个像素的锐化程度
    cv::Mat diff = sharpened - src;
    diff = diff.mul(mask3ch);  // 逐像素乘以掩码
    dst = src + diff;
}
```

## 总结

### amount（锐化强度）
- **控制**：锐化的强度
- **范围**：0.0 - 2.0（推荐1.0-1.5）
- **效果**：值越大，边缘越锐利，细节越明显
- **副作用**：过大会产生光晕和过度锐化

### threshold（边缘阈值）
- **控制**：哪些区域被锐化
- **范围**：0 - 50（推荐15-25）
- **效果**：值越大，锐化越有选择性，只锐化强边缘
- **优势**：避免平滑区域噪声放大，减少光晕

### 黄金组合
```
内窥镜图像：amount=1.5, threshold=20
高对比度图像：amount=1.0, threshold=30
```

### 调参口诀
1. 先调threshold找到合适的锐化区域
2. 再调amount控制锐化强度
3. 有光晕 → 增加threshold
4. 效果弱 → 降低threshold或增加amount
