# EndoscopeSystem — 开发指南

## 项目目标

在野火 STM32MP157 开发板上，使用 OV5640 摄像头 + LCD 屏幕，开发一个**单机内窥镜图像处理系统**。

先在虚拟机 Ubuntu 上完成全部软件开发（使用视频文件模拟摄像头），再交叉编译部署到开发板。

## 硬件环境

| 硬件 | 规格 |
|------|------|
| 开发板 | 野火 STM32MP157 (Cortex-A7 双核 650MHz, 1GB DDR3) |
| 摄像头 | OV5640 (DCMI 接口, stm32-dcmi 驱动) |
| 屏幕 | 800x480 LCD (/dev/fb0) |
| 采集参数 | YUYV 640x480 @30fps (推荐) |

## 开发环境

| 环境 | 用途 |
|------|------|
| Windows 主机 | 宿主机，运行 VMware |
| Ubuntu 虚拟机 (VMware) | 软件开发、编译、调试 |
| STM32MP157 开发板 | 最终部署运行 |

## 参考项目

本项目以两个开源项目为参考重构：

| 项目 | 借鉴内容 |
|------|---------|
| VideoMonitor | Qt GUI 框架、OpenCV 图像处理管线、QImage↔Mat 转换、拍照/录像框架 |
| cap-v4l2-master | 健壮的 V4L2 初始化流程、select 等待、能力查询、曝光/翻转控制 |

### 与参考项目的关键区别

| | VideoMonitor | cap-v4l2 | **EndoscopeSystem (本项目)** |
|--|-------------|----------|---------------------------|
| 架构 | C/S 网络架构 | 单机命令行 | **单机 Qt GUI** |
| 采集 | 简单 V4L2 + libjpeg | 完整 V4L2 + select | **完整 V4L2 + OpenCV cvtColor** |
| 显示 | TCP 传输后显示 | 无实时显示 | **本地直接显示** |
| 图像处理 | 边缘检测/锐化/阈值 | 无 | **8 个算法 (含内窥镜专用)** |
| 输入源 | 仅 V4L2 | 仅 V4L2 | **抽象接口 (V4L2 + 文件)** |
| 线程安全 | volatile bool | 无线程 | **std::atomic + Qt 信号槽** |
