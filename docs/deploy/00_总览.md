## ⚠️ 注意事项

1. **必须用 Linaro gcc 7.5.0 交叉编译**，不能用 apt 装的交叉编译器（glibc 版本不兼容）
2. **不要在 VM_Share 共享目录下编译**，会很慢且可能出错，编译在 /home/chris/ 下进行
3. **开发板 root SSH 被禁止**，必须先用 debian 用户登录再 su - root
4. **WiFi 连接延迟较大**（40~467ms），NFS 传输大文件时可能较慢
