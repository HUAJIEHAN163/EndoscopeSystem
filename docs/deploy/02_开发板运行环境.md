| 项目 | 值 |

| 延迟 | 40~467ms（WiFi 波动较大） |

## 2.3 SSH 登录方式

```bash
# 第一步：用 debian 用户 SSH 登录（root SSH 被禁止）
ssh debian@192.168.0.43
# 密码：temppwd

# 第二步：切换到 root
su - root
# 密码：root
```

注意：输入密码时屏幕不会显示任何字符，直接盲打回车。

## 2.4 摄像头

| 项目 | 值 |
|------|-----|
| 型号 | OV5640 |
| 设备节点 | /dev/video0 |
| 最大分辨率 | 2592x1944（500 万像素） |

查看摄像头参数：
```bash
v4l2-ctl -d /dev/video0 --list-ctrls
```

## 2.6 NFS 客户端

```bash
# 安装（首次需要）
apt install nfs-common

# 挂载虚拟机共享目录
mount -t nfs 192.168.0.130:/home/chris/nfs_share /mnt/nfs -o nolock
```

挂载点：`/mnt/nfs`
