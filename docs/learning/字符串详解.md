# C/C++/Qt 字符串详解

## 一、三种字符串对比

| | C 字符串 | C++ std::string | Qt QString |
|---|---|---|---|
| 头文件 | `<string.h>` 或 `<cstring>` | `<string>` | `<QString>` |
| 本质 | char 数组 + '\0' 结尾 | 封装了 char 数组的类 | 封装了 Unicode 的类 |
| 内存管理 | 手动（容易溢出） | 自动 | 自动 |
| 编码 | ASCII/UTF-8（取决于环境） | ASCII/UTF-8 | Unicode（支持中文） |
| 使用场景 | 嵌入式底层、系统调用参数 | 通用 C++ 程序 | Qt 程序 |
| 我们项目中 | ioctl 参数、设备路径 | V4l2Capture 构造函数参数 | UI 显示、日志、文件路径 |

## 二、C 字符串（char 数组）

### 2.1 声明和初始化

```cpp
// 字符串字面量（存在只读区，不能修改）
const char *s1 = "hello";

// 字符数组（可修改）
char s2[] = "hello";           // 编译器自动算长度，等价于 char s2[6]
char s3[32] = "hello";         // 32 字节空间，前 6 字节是 "hello\0"
char s4[32] = {0};             // 32 字节全部清零（空字符串）
char s5[32];                   // 未初始化（内容随机，危险）

// 内存中的样子
// s2: ['h']['e']['l']['l']['o']['\0']
//                                ↑ 结尾标记，strlen 靠它判断长度
```

### 2.2 常用操作

```cpp
#include <string.h>   // C 字符串函数
// 或
#include <cstring>    // C++ 中推荐用这个

char buf[64];

// 复制
strcpy(buf, "hello");              // buf = "hello"（危险：不检查溢出）
strncpy(buf, "hello", sizeof(buf)); // 安全版本，限制最大长度

// 拼接
strcat(buf, " world");             // buf = "hello world"（危险）
strncat(buf, " world", sizeof(buf) - strlen(buf) - 1);  // 安全版本

// 长度
int len = strlen(buf);             // 11（不含 '\0'）

// 比较
if (strcmp(s1, s2) == 0) { }       // 相等返回 0（不能用 == 比较！）
if (strncmp(s1, s2, 3) == 0) { }  // 只比较前 3 个字符

// 查找
char *p = strstr(buf, "world");    // 找子串，返回指针，找不到返回 NULL
char *p = strchr(buf, 'w');        // 找单个字符

// 格式化
sprintf(buf, "FPS: %.1f", 21.7);   // buf = "FPS: 21.7"（危险：不检查溢出）
snprintf(buf, sizeof(buf), "FPS: %.1f", 21.7);  // 安全版本
```

### 2.3 作为函数参数

```cpp
// 传入（只读）：const char*
void print(const char *msg) {
    printf("%s\n", msg);
}
print("hello");           // 字符串字面量
print(buf);               // char 数组

// 传出（写入）：char* + 长度
void getName(char *out, int maxLen) {
    strncpy(out, "test", maxLen);
}
char name[32];
getName(name, sizeof(name));

// 返回值：通常不返回局部数组（会悬空），返回 const char* 指向静态/全局数据
const char *getVersion() {
    return "1.0.0";        // 字符串字面量在只读区，不会悬空
}
```

### 2.4 我们项目中的使用

```cpp
// V4L2 设备路径
m_device = "/dev/video0";                    // std::string，但底层是 char*
open(m_device.c_str(), O_RDWR);              // .c_str() 转成 const char*

// ioctl 结构体中的字符串
qDebug() << "Driver:" << (char*)cap.driver;  // V4L2 返回的是 char 数组

// strerror：系统错误码转字符串
emit errorOccurred(QString("错误: %1").arg(strerror(errno)));
```

### 2.5 常见陷阱

```cpp
// 陷阱 1：忘记 '\0'
char buf[5];
strncpy(buf, "hello", 5);   // buf = ['h','e','l','l','o'] 没有 '\0'！
// 后续 strlen(buf) 会越界读取直到碰到 '\0'

// 陷阱 2：用 == 比较
if (s1 == s2)     // ❌ 比较的是指针地址，不是内容
if (strcmp(s1, s2) == 0)  // ✅ 比较内容

// 陷阱 3：返回局部数组
char *bad() {
    char buf[32] = "hello";
    return buf;    // ❌ buf 是局部变量，函数返回后内存被回收
}

// 陷阱 4：缓冲区溢出
char buf[8];
strcpy(buf, "this is a very long string");  // ❌ 溢出，覆盖其他内存
```

## 三、C++ std::string

### 3.1 声明和初始化

```cpp
#include <string>

std::string s1;                    // 空字符串 ""
std::string s2 = "hello";         // 从字面量初始化
std::string s3("hello");          // 同上
std::string s4 = s2;              // 拷贝
std::string s5(10, 'x');          // "xxxxxxxxxx"（10 个 x）
```

### 3.2 常用操作

```cpp
std::string s = "hello";

// 拼接（最大优势：不用担心溢出）
s += " world";                     // s = "hello world"
std::string s2 = s + "!";         // s2 = "hello world!"
s.append(" end");                  // s = "hello world end"

// 长度
int len = s.length();              // 或 s.size()，两者等价

// 比较（可以直接用 ==）
if (s == "hello world end") { }    // ✅ 直接比较内容
if (s != "other") { }
if (s < "z") { }                   // 字典序比较

// 查找
size_t pos = s.find("world");      // 返回位置，找不到返回 std::string::npos
if (pos != std::string::npos) {
    // 找到了，pos = 6
}

// 截取
std::string sub = s.substr(6, 5); // 从位置 6 开始取 5 个字符 → "world"

// 替换
s.replace(6, 5, "earth");         // "hello earth end"

// 清空
s.clear();                         // s = ""
bool empty = s.empty();            // true

// 转 C 字符串
const char *cstr = s.c_str();     // 返回 const char*，用于系统调用
```

### 3.3 数字和字符串互转

```cpp
// 数字 → string
std::string s1 = std::to_string(42);       // "42"
std::string s2 = std::to_string(3.14);     // "3.140000"

// string → 数字
int n = std::stoi("42");                    // 42
double d = std::stod("3.14");              // 3.14
```

### 3.4 作为函数参数

```cpp
// 传入（只读）：const std::string& （引用，不拷贝）
void print(const std::string &msg) {
    std::cout << msg << std::endl;
}
print("hello");                    // 字面量自动转 string
print(s);                          // 传引用，不拷贝

// 传入（需要修改）：std::string&
void appendWorld(std::string &s) {
    s += " world";
}

// 返回值：直接返回（编译器会优化，不会真的拷贝）
std::string getName() {
    return "test";                 // ✅ 安全，不像 C 字符串会悬空
}
```

### 3.5 我们项目中的使用

```cpp
// V4l2Capture 构造函数
V4l2Capture(const std::string &device = "/dev/video0", ...);

// 存储设备路径
std::string m_device;              // 成员变量
m_device = device;                 // 赋值

// 传给系统调用时转 C 字符串
m_fd = open(m_device.c_str(), O_RDWR);

// cv::imwrite 也需要 std::string
cv::imwrite(filename.toStdString(), mat);
//                   ↑ QString 转 std::string
```

## 四、Qt QString

### 4.1 声明和初始化

```cpp
#include <QString>

QString s1;                        // 空字符串 ""
QString s2 = "hello";             // 从字面量初始化
QString s3("hello");              // 同上
QString s4 = s2;                  // 拷贝（隐式共享，实际不拷贝数据）
QString s5 = QString::number(42); // 从数字创建 "42"
```

### 4.2 常用操作

```cpp
QString s = "hello";

// 拼接
s += " world";                     // s = "hello world"
QString s2 = s + "!";

// 长度
int len = s.length();

// 比较
if (s == "hello world") { }        // ✅ 直接比较

// 查找
int pos = s.indexOf("world");      // 6，找不到返回 -1
bool has = s.contains("world");    // true

// 截取
QString sub = s.mid(6, 5);        // "world"（从位置 6 取 5 个）
QString left = s.left(5);         // "hello"（取前 5 个）
QString right = s.right(5);       // "world"（取后 5 个）

// 替换
s.replace("world", "earth");      // "hello earth"

// 分割
QString csv = "a,b,c,d";
QStringList parts = csv.split(","); // ["a", "b", "c", "d"]

// 合并（QStringList）
QString joined = parts.join("+");  // "a+b+c+d"

// 去除末尾字符
s.chop(1);                         // 去掉最后 1 个字符

// 清空
s.clear();
bool empty = s.isEmpty();          // true（注意是 isEmpty 不是 empty）

// 大小写
s = "Hello";
s.toUpper();                       // "HELLO"
s.toLower();                       // "hello"
```

### 4.3 格式化（arg 占位符）

```cpp
// %1 %2 %3 是占位符，.arg() 按顺序替换
QString s1 = QString("FPS: %1").arg(21.7);
// → "FPS: 21.7"

QString s2 = QString("%1x%2 @ %3fps").arg(640).arg(480).arg(30);
// → "640x480 @ 30fps"

// 控制格式
QString s3 = QString("FPS: %1").arg(21.7, 0, 'f', 1);
//                                    ↑值  ↑宽度(0=自动) ↑格式(f=小数) ↑小数位数
// → "FPS: 21.7"

// 十六进制
QString s4 = QString("0x%1").arg(255, 4, 16, QChar('0'));
// → "0x00ff"
```

### 4.4 类型转换

```cpp
// QString → 数字
int n = s.toInt();                  // "42" → 42
double d = s.toDouble();           // "3.14" → 3.14

// 数字 → QString
QString s1 = QString::number(42);          // "42"
QString s2 = QString::number(3.14, 'f', 1); // "3.1"

// QString → std::string
std::string std_s = s.toStdString();

// std::string → QString
QString q_s = QString::fromStdString(std_s);
// 或
QString q_s2 = QString(std_s.c_str());

// QString → const char*（临时使用）
qDebug() << s.toUtf8().constData();
// 或直接用 qDebug() << s;  （qDebug 原生支持 QString）

// const char* → QString
QString s3 = QString("hello");             // 自动转换
QString s4 = QString::fromUtf8(cstr);      // 明确指定 UTF-8 编码
```

### 4.5 作为函数参数

```cpp
// 传入（只读）：const QString&
void showStatus(const QString &msg) {
    m_lblStatus->setText(msg);
}
showStatus("就绪");                 // 字面量自动转 QString
showStatus(QString("FPS: %1").arg(fps));

// 返回值：直接返回（QString 隐式共享，返回不拷贝数据）
QString getFilename() {
    return QString("%1/captures/%2.jpg")
        .arg(m_savePath)
        .arg(QDateTime::currentDateTime().toString("yyyyMMdd_hhmmss"));
}
```

### 4.6 我们项目中的高频用法

```cpp
// 1. 状态栏显示
m_lblStatus->setText("调试模式");
m_lblStatus->setText(QString("拍照: %1").arg(filename));

// 2. FPS 显示
m_lblFps->setText(QString("FPS: %1").arg(fps, 0, 'f', 1));

// 3. 调试日志
qDebug() << QString("[CAP 帧%1] YUYV→BGR:%2ms push:%3ms")
            .arg(frameCount).arg(tYuv2Bgr).arg(tPush);

// 4. 文件路径拼接
QString filename = QString("%1/captures/%2.jpg")
    .arg(m_savePath)
    .arg(QDateTime::currentDateTime().toString("yyyyMMdd_hhmmss"));

// 5. 配置文件读取
QString device = settings.value("camera/device", "/dev/video0").toString();

// 6. 传给 OpenCV（需要转 std::string）
cv::imwrite(filename.toStdString(), mat);

// 7. 传给系统调用（需要转 const char*）
m_source = new FileCapture(testFile.toStdString(), 30, this);
```

## 五、三种字符串互转速查

```
const char* ←→ std::string ←→ QString

const char* → std::string:
  std::string s(cstr);
  std::string s = cstr;

std::string → const char*:
  const char *c = s.c_str();

const char* → QString:
  QString q = QString(cstr);
  QString q = QString::fromUtf8(cstr);

QString → const char*:
  const char *c = q.toUtf8().constData();  // 临时使用

std::string → QString:
  QString q = QString::fromStdString(s);

QString → std::string:
  std::string s = q.toStdString();
```

转换关系图：

```
const char*  ←──c_str()──  std::string  ←──toStdString()──  QString
     │                          ↑                               ↑
     └──── 直接赋值 ────────────┘                               │
     └──── QString(cstr) ──────────────────────────────────────┘
```

## 六、选择哪种字符串

| 场景 | 用哪种 | 原因 |
|---|---|---|
| 系统调用参数（open/ioctl） | const char* | 系统调用只接受 C 字符串 |
| 类的成员变量（存储路径等） | std::string | 自动内存管理，不依赖 Qt |
| Qt UI 显示（setText/qDebug） | QString | Qt 控件只接受 QString |
| 日志输出 | QString | qDebug 原生支持，arg 格式化方便 |
| OpenCV 函数参数 | std::string | OpenCV API 用 std::string |
| 临时拼接后传给系统调用 | QString → .toStdString() → .c_str() | 用 QString 拼接方便，最后转一下 |
