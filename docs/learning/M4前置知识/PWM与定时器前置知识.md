# PWM 与定时器前置知识

## 一、定时器基础

### 1.1 定时器的本质

定时器 = 一个硬件计数器，按照时钟频率自动递增，溢出时触发事件（中断、PWM 翻转等）。

```
时钟源 (64MHz)
    │
    ▼ ÷ (PSC+1)
预分频器 ──→ 计数器时钟 (1MHz)
                │
                ▼ 计数 0→ARR
            计数器 ──→ 溢出时触发事件
```

### 1.2 关键参数

| 参数 | 寄存器 | 含义 |
|---|---|---|
| PSC（Prescaler） | 预分频器 | 时钟分频系数 = PSC + 1 |
| ARR（Auto-Reload） | 自动重装值 | 计数器从 0 数到 ARR 后溢出 |
| CCR（Capture/Compare） | 比较值 | PWM 模式下控制占空比 |

### 1.3 频率计算公式

```
定时器时钟 = 系统时钟 / (PSC + 1)
溢出频率   = 定时器时钟 / (ARR + 1)
溢出周期   = 1 / 溢出频率
```

**本项目示例**（TIM2 PWM，系统时钟 64MHz）：
```
定时器时钟 = 64MHz / (63+1) = 1MHz
PWM 频率   = 1MHz / (999+1) = 1kHz
PWM 周期   = 1ms
```

**本项目示例**（TIM3 定时中断，系统时钟 64MHz）：
```
定时器时钟 = 64MHz / (6399+1) = 10kHz
中断频率   = 10kHz / (99+1) = 100Hz
中断周期   = 10ms
```

## 二、PWM 输出

### 2.1 什么是 PWM

PWM（Pulse Width Modulation）= 脉冲宽度调制。通过改变高电平占比（占空比）来控制平均输出功率。

```
占空比 25%:  ██        ██        ██        ██
            ────────  ────────  ────────  ────────

占空比 75%:  ██████    ██████    ██████    ██████
            ────────  ────────  ────────  ────────

占空比 100%: ████████  ████████  ████████  ████████
```

### 2.2 占空比计算

```
占空比 = CCR / (ARR + 1) × 100%

本项目：ARR = 999
  CCR = 0    → 0%（LED 灭）
  CCR = 500  → 50%（半亮）
  CCR = 999  → 99.9%（最亮）
```

### 2.3 HAL 库 PWM 操作

```c
// 启动 PWM 输出
HAL_TIM_PWM_Start(&htim2, TIM_CHANNEL_3);

// 动态修改占空比（不需要停止 PWM）
__HAL_TIM_SET_COMPARE(&htim2, TIM_CHANNEL_3, 500);  // 50%

// 停止 PWM
HAL_TIM_PWM_Stop(&htim2, TIM_CHANNEL_3);
```

### 2.4 PWM 频率选择

| 应用 | 推荐频率 | 原因 |
|---|---|---|
| LED 调光 | 1kHz+ | 人眼无法感知闪烁（>100Hz 即可） |
| 电机控制 | 10-20kHz | 避免可听噪声（人耳范围 20Hz-20kHz） |
| 舵机控制 | 50Hz | 舵机标准信号频率 |

本项目 LED 调光用 1kHz，足够平滑。

## 三、定时器中断

### 3.1 工作原理

计数器从 0 数到 ARR 后溢出，触发更新中断（Update Interrupt），然后自动从 0 重新开始。

```
计数器值:  0 → 1 → 2 → ... → ARR → 0 → 1 → ...
                                  ↑
                              触发中断
```

### 3.2 HAL 库定时器中断

```c
// 启动定时器中断
HAL_TIM_Base_Start_IT(&htim3);

// 中断回调函数（用户实现）
void HAL_TIM_PeriodElapsedCallback(TIM_HandleTypeDef *htim)
{
    if (htim->Instance == TIM3) {
        // TIM3 溢出，每 10ms 执行一次
        // 在这里做按键轮询
    }
}
```

### 3.3 中断回调 vs 中断处理函数

```
硬件中断触发
    │
    ▼
TIM3_IRQHandler()              ← stm32mp1xx_it.c（CubeMX 生成，不要改）
    │
    ▼
HAL_TIM_IRQHandler(&htim3)     ← HAL 库内部处理
    │
    ▼
HAL_TIM_PeriodElapsedCallback() ← 用户实现（在 main.c 的 USER CODE 中）
```

## 四、本项目中的实际应用

### 4.1 TIM2 — LED PWM 调光

| 参数 | 值 | 说明 |
|---|---|---|
| 引脚 | PB10 (TIM2_CH3, AF1) | 直驱 HW-269 LED |
| PSC | 63 | 64MHz / 64 = 1MHz |
| ARR | 999 | 1MHz / 1000 = 1kHz |
| CCR 范围 | 0-999 | 0%~99.9% 占空比 |

A7 发送 `L50\n` → M4 解析 → `CCR = 50 × 999 / 100 = 499`

### 4.2 TIM3 — 按键轮询定时器

| 参数 | 值 | 说明 |
|---|---|---|
| PSC | 6399 | 64MHz / 6400 = 10kHz |
| ARR | 99 | 10kHz / 100 = 100Hz (10ms) |
| 中断 | 使能 | 每 10ms 触发回调 |

在回调中读取 KEY1/KEY2 的 GPIO 电平，检测上升沿 + 消抖。

## 五、常见面试问题

### Q1：如何用定时器产生精确的 1 秒延时？

配置 PSC 和 ARR 使溢出周期 = 1 秒。例如 64MHz 时钟：PSC=63999, ARR=999 → 64MHz/64000/1000 = 1Hz。

### Q2：PWM 的死区时间是什么？

互补 PWM 输出中，上下桥臂切换时需要短暂的死区（两个都关断），防止直通短路。用于电机驱动的 H 桥电路。本项目不涉及。

### Q3：输入捕获（Input Capture）是什么？

定时器的另一种模式：当引脚电平变化时，自动记录当前计数器值。用于测量脉冲宽度、频率。例如测量超声波回波时间。
