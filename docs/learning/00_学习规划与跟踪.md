### 各层定位

| 层级 | 学习载体 | 核心目标 |
|------|---------|---------|
| A7 应用层 | 当前内窥镜项目 | 掌握 C++/Qt/OpenCV/V4L2，能独立开发嵌入式 Linux 图像应用 |
| M4 实时层 | 后续 M4 子项目 | 掌握 C/HAL/RTOS，能开发裸机外设驱动和实时控制程序 |
| 通用技能 | 贯穿两个项目 | Git 版本管理、Linux 操作与调试、通信协议理解 |

### 通信协议为什么是通用技能

同一个协议在两侧都会用到，理解协议本身是通用的，实现方式不同：

| 协议 | A7 侧（Linux 应用层） | M4 侧（裸机/RTOS） |
|------|----------------------|-------------------|
| UART | /dev/ttySTM0 + termios | HAL_UART_Transmit/Receive |
| I2C | /dev/i2c-x + ioctl | HAL_I2C_Master_Transmit |
| SPI | /dev/spidev + ioctl | HAL_SPI_TransmitReceive |

协议原理学一次，两侧都能用。动手练习主要在 M4 阶段。

## 二、A7 应用层 — 学习方法与目标

### 2.1 C++

**学习方法**：对照项目代码学，遇到不懂的语法当场查，不要先系统学完再看代码。

**学习路径**：
1. 类与对象：看 `MainWindow` 类的定义和使用
2. 继承：`MainWindow : public QMainWindow`、`VideoSource : public QThread`
3. 虚函数与重写：`paintEvent`、`keyPressEvent` 的 override
4. 构造/析构函数：`MainWindow::MainWindow()`、`~MainWindow()`
5. 指针与 new/delete：Qt 的父子内存管理机制
6. 引用与 const：函数参数中的 `const QImage &image`
7. STL 容器：`std::deque`（帧缓存）、`std::vector`（OpenCV 通道操作）

**目标要求**：
- [ ] 能看懂项目中所有 C++ 语法
- [ ] 能独立写一个带继承和虚函数的类
- [ ] 理解指针、引用、const 的区别和使用场景
- [ ] 理解 new/delete 和 Qt 父子内存管理

**参考资源**：C++ Primer（遇到什么查什么，不要从头读）

### 2.2 Qt

**学习方法**：从 `mainwindow.cpp` 入手，逐个理解用到的 Qt 类和机制。

**学习路径**：
1. 信号槽机制：`connect` 的写法和原理
2. 控件创建：QPushButton、QCheckBox、QSlider、QLabel、QComboBox
3. 布局管理：QVBoxLayout、QHBoxLayout、QGroupBox、QStackedWidget
4. 事件机制：paintEvent、keyPressEvent、update()
5. 多线程：QThread、跨线程信号槽（QueuedConnection）
6. 文件操作：QSettings（配置文件）、QJsonDocument（预设文件）
7. 定时器：QTimer

**目标要求**：
- [ ] 能解释信号槽的工作原理和跨线程通信
- [ ] 能手写一个带布局、按钮、信号槽的简单界面
- [ ] 理解 Qt 事件循环（app.exec）和绘制机制（update → paintEvent）
- [ ] 理解 Qt 父子内存管理机制

**参考资源**：Qt 官方文档 https://doc.qt.io（搜类名即可）

### 2.3 OpenCV

**学习方法**：对照 `imageprocessor.cpp` 逐个算法学，先理解算法原理再看 API。

**学习路径**：
1. Mat 基础：创建、像素访问、类型（CV_8UC3、CV_32FC3）
2. 色彩空间：cvtColor（BGR/RGB/Lab/Gray 互转）
3. 通道操作：split、merge
4. 滤波：GaussianBlur（锐化用）、bilateralFilter（降噪用）
5. 形态学：erode（去雾暗通道用）
6. 直方图：CLAHE 自适应均衡化
7. 边缘检测：Canny
8. 阈值处理：threshold
9. 相机标定：initUndistortRectifyMap、remap

**目标要求**：
- [ ] 能解释 CLAHE、去雾（暗通道先验）、白平衡（灰度世界法）的原理
- [ ] 能解释 Mat 的内存布局和常见类型
- [ ] 能独立调用 OpenCV API 实现一个图像处理功能
- [ ] 理解畸变校正的数学模型（相机内参、畸变系数）

**参考资源**：《Learning OpenCV 3》Ch 2-3、5、10-14、18

### 2.4 V4L2

**学习方法**：逐行读 `v4l2capture.cpp`，对照 V4L2 API 文档理解每个 ioctl。

**学习路径**：
1. 设备打开：open("/dev/video0")
2. 能力查询：VIDIOC_QUERYCAP
3. 格式设置：VIDIOC_S_FMT（YUYV、分辨率）
4. Buffer 管理：VIDIOC_REQBUFS → VIDIOC_QUERYBUF → mmap
5. 采集循环：VIDIOC_QBUF → VIDIOC_DQBUF → 处理 → VIDIOC_QBUF
6. 流控制：VIDIOC_STREAMON / VIDIOC_STREAMOFF

**目标要求**：
- [ ] 能画出 V4L2 采集的完整流程图
- [ ] 能解释 mmap 的作用（用户空间直接访问内核 buffer，零拷贝）
- [ ] 能解释 YUYV 格式和 RGB 的区别
- [ ] 理解 buffer 队列机制（QBUF/DQBUF 轮转）

**参考资源**：V4L2 官方文档 https://www.kernel.org/doc/html/latest/userspace-api/media/v4l/v4l2.html

## 三、M4 实时层 — 学习方法与目标

### 3.1 C 语言巩固

**学习方法**：通过 HAL 库代码巩固，重点关注指针、结构体、位操作。

**目标要求**：
- [ ] 熟练使用指针操作寄存器
- [ ] 熟练使用结构体（HAL 库大量使用）
- [ ] 熟练使用位操作（寄存器配置）
- [ ] 理解 volatile 关键字（硬件寄存器必须用）

### 3.2 HAL 库 + CubeIDE

**学习方法**：用 CubeMX 生成初始化代码，逐个外设练习。

**学习路径**（按顺序）：
1. GPIO：LED 闪烁、按键检测（入门第一步）
2. UART：串口收发、printf 重定向（调试必备）
3. TIM：定时器中断、PWM 输出（LED 补光灯控制）
4. I2C：读写传感器（温湿度、OLED）
5. SPI：Flash 读写、TFT 屏驱动
6. ADC：模拟信号采集（温度监控）
7. DMA：配合 UART/SPI 实现不占 CPU 的数据搬运
8. 中断：NVIC 优先级管理

**目标要求**：
- [ ] 能用 CubeMX 配置引脚和时钟，生成初始化代码
- [ ] 能独立实现 GPIO + UART + TIM 的基础功能
- [ ] 能用 I2C 读写一个传感器
- [ ] 能用 PWM 控制 LED 亮度（内窥镜补光灯场景）
- [ ] 理解中断优先级和响应机制

### 3.3 FreeRTOS

**学习方法**：在 CubeIDE 中开启 FreeRTOS，从双任务开始练。

**学习路径**：
1. 任务创建与调度
2. 信号量（二值、计数）
3. 消息队列
4. 软件定时器
5. 任务优先级与抢占

**目标要求**：
- [ ] 能创建多个任务并理解调度原理
- [ ] 能用信号量实现任务同步
- [ ] 能用消息队列实现任务间通信
- [ ] 理解优先级反转问题

### 3.4 A7-M4 核间通信

**学习方法**：结合项目 P2（按键控制），实现 M4 发指令、A7 执行。

**目标要求**：
- [ ] 能通过 UART 实现 A7-M4 双向通信
- [ ] 了解 OpenAMP/RPMsg 的基本概念

## 四、通用技能 — 学习方法与目标

### 4.1 Git

**学习方法**：在当前项目中持续使用。

**目标要求**：
- [ ] 熟练使用 add、commit、push、pull
- [ ] 能创建和合并分支（feature/bugfix）
- [ ] 能解决合并冲突
- [ ] 理解 .gitignore 的作用

### 4.2 Linux

**学习方法**：边做项目边积累，遇到问题查 man 手册。

**目标要求**：
- [ ] 熟练使用基础命令（文件、进程、网络）
- [ ] 能写简单的 Shell 脚本（构建、部署）
- [ ] 能用 dmesg 查内核日志排查硬件问题
- [ ] 能用 top/htop 分析 CPU 和内存占用
- [ ] 能配置 systemd 服务实现开机自启
- [ ] 理解 /dev、/proc、/sys 的作用

### 4.3 通信协议

**学习方法**：先学协议原理（波形、时序），再在 M4 上动手实现，最后在 A7 上用用户空间接口操作。

**目标要求**：
- [ ] 能画出 UART、I2C、SPI 的时序图
- [ ] 能说清三者的区别（速度、线数、主从、全双工/半双工）
- [ ] 能在 M4 上用 HAL 库实现三种协议的收发
- [ ] 能在 A7 上通过 /dev 设备节点操作串口和 I2C

### 4.4 医疗行业知识（了解即可）

- [ ] 知道 IEC 62304（医疗软件生命周期标准）的基本要求
- [ ] 知道 IEC 60601（医疗电气安全标准）的存在
- [ ] 理解软件版本管理、文档记录、测试追溯在医疗行业的重要性
